Animate

The animate() method enables custom animations.

The required params parameter defines the CSS properties to be animated.

The optional speed parameter specifies the duration of the effect. It can take the following values: "slow", "fast", or milliseconds.

The optional callback parameter is a function to be executed after the animation has completed.

Example: animate()


      $(document).ready(function(){
        $("#btn1").click(function() {
          $("#div1").animate({
            left: '200px'
          });
        });
      });

Example: animate() multiple properties


      $(document).ready(function(){
        $("#btn2").click(function() {
          $("#div2").animate({
            left: '200px',
            width: '200px',
            opacity: '0.5'
          });
        });
      });

Example: animate() relative values


      $(document).ready(function(){
        $("#btn3").click(function() {
          $("#div3").animate({
            left: '200px',
            width: '+=100px',
            opacity: '-=0.3'
          });
        });
      });

Example: animate() + toggle


      $(document).ready(function(){
        $("#btn4").click(function() {
          $("#div4").animate({
            height: 'toggle'
          });
        });
      });

Example: animate() chained


      $(document).ready(function(){
        $("#btn5").click(function() {
          var div = $("#div5");
          div.animate({
            left: '200px'
          }, 'slow');
          div.animate({
            width: '300px'
          }, 'slow');
          div.animate({
            opacity: '-=0.5'
          }, 'slow');
        });
      });

Stop Animations

The stop() method is used to stop animations and effects before they are finished.

The stop() method works for all jQuery effect functions, including sliding, fading, and custom animations.

The optional stopAll parameter specifies whether or not to clear the animation queue. Default is false, which means that only the active animation will be stopped, allowing any queued animations to be performed afterwards.

The optional goToEnd parameter specifies whether or not to complete the current animation immediately. Default is false.

By default, the stop() method stops the current animation being performed on the selected element.

Example: stop()

Slide Down Panel
Hello There!

      $(document).ready(function(){
        $("#div7").click(function() {
          $("#panel7").slideDown(5000);
        });
        $("#stop").click(function() {
          $("#panel7").stop();
        });
      });

Example: stop() with parameters

The "Start" button starts the animation.

The "Stop" button stops the current active animation, but allows the queued animations to be performed afterwards.

The "Stop all" button stops the current active animation and clears the animation queue; so all animations on the element are stopped.

The "Stop but finish" button rushes through the current active animation, then it stops.

      $(document).ready(function(){
        $("#start").click(function() {
          $("#div8").animate({
            left: '200px'
          }, 5000);
          $("#div8").animate({
            fontSize: '30px'
          }, 5000);
        });

        $("#stop1").click(function() {
          $("#div8").stop();
        });
        $("#stop2").click(function() {
          $("#div8").stop(true);
        });
        $("#stop3").click(function() {
          $("#div8").stop(true, true);
        });
      });
