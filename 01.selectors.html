Selectors

jQuery selectors allow you to select and manipulate HTML elements.

jQuery selectors are used to "find" (or select) HTML elements based on their name, id, classes, types, attributes, values of attributes and much more. They are based on the existing CSS Selectors, and in addition, has some of its own custom selectors. All jQuery selectors start with a $ sign and parenthesis.

Example: hide() using element

Hide all div using element:

This is a div-element.
This is another div-element.

      $(document).ready(function(){
        $("#btn1").click(function() {
          $("div").hide();
        });
      });

Example: hide() using element's id

Hide all p-elements using element's id

This is a paragraph

This is another paragraph.


      $(document).ready(function(){
        $("#btn2").click(function() {
          $("#p1, #p2").hide();
        });
      });

Example: hide() using element's class

Hide the divs using element's class:

This is a div.
This is another div.

      $(document).ready(function(){
        $("#btn3").click(function() {
          $(".test1, .test2").hide();
        });
      });

Example: hide() the current element (the button) using "this":

Hide the button using "this":


      $(document).ready(function(){
        $("#btn4").click(function() {
          $(this).hide();
        });
      });
