Fade

The fade() method can be used to fade an element in and out of visibility.

The optional speed parameter specifies the speed of the hiding and showing, and can take the following values: "slow", "fast", or milliseconds.

The optional callback parameter is a function to be executed after the hide or show completes.

Note: the examples on this page use different speed parameters.

Example: fadeIn()

Fade in the boxes using the fadeIn() method:


    $(document).ready(function(){
      $("#btn1").click(function() {
        $("#div1").fadeIn();
        $("#div2").fadeIn("slow");
        $("#div3").fadeIn(3000);
      });
    });

Example: fadeOut()

Fade out the boxes using the fadeOut() method:


    $(document).ready(function(){
      $("#btn2").click(function() {
        $("#div4").fadeOut();
        $("#div5").fadeOut("slow");
        $("#div6").fadeOut(3000);
      });
    });

Example: fadeToggle()

Click the button (multiple times) to fade the boxes in and out (toggle):


    $(document).ready(function(){
      $("#btn3").click(function() {
        $("#div7").fadeToggle();
        $("#div8").fadeToggle("slow");
        $("#div9").fadeToggle(3000);
      });
    });

Example: fadeTo()

The fadeTo() method can be used to fade an element to a specified opacity:


    $(document).ready(function(){
      $("#btn4").click(function() {
        $("#div10").fadeTo("slow", 0.15);
        $("#div11").fadeTo("slow", 0.30);
        $("#div12").fadeTo("slow", 0.60);
      });
    });
