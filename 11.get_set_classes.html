Get and Set CSS Classes

The following methods can be used to manipulate CSS classes:

  • addClass() - adds one or more classes to the selected elements
  • removeClass() - removes one or more classes from the selected elements
  • toggleClass() - toggles between adding and removing classes from the selected elements
  • css() - sets or returns the style attribute

Example: addClass()

Use the addClass() method to add classes to the selected elements:

      $(document).ready(function(){
        $("#btn1").click(function() {
          $("#div1").addClass("blue");
          $("#test").addClass("yellow");
          $("#test2").addClass("big");
        });
      });

Example: addClass() multiple classes

Use the addClass() method with parameters to add multiple classes to the selected element:



This is some text inside a div! 

      $(document).ready(function(){
        $("#btn2").click(function() {
          $("#div2").addClass("big green");
        });
      });

Example: removeClass()

Use the removeClass() method to remove classes from the selected elements:



This is a paragraph inside a div!


      $(document).ready(function(){
        $("#btn3").click(function() {
          $("#div3").removeClass("blue");
          $("#test3").removeClass("yellow");
        });
      });

Example: toggleClass()

Use the toggleClass() method to toggle the class of the selected element:



This is a paragraph inside a div!


      $(document).ready(function(){
        $("#btn4").click(function() {
          $("#div4").toggleClass("blue");
          $("#test4").toggleClass("yellow big");
        });
      });
