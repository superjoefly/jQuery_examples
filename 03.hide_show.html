Hide and Show

The hide() and show() methods can be used to hide and show an element.

The optional speed parameter specifies the speed of the hiding and showing, and can take the following values: "slow", "fast", or milliseconds.

The optional callback parameter is a function to be executed after the hide or show completes.

Example: hide()

The hide() method can be used to hide an element:


      $(document).ready(function(){
        $("#btn1").click(function() {
          $("#div1").hide("slow");
        });
      });

Example: show()

The show() method can be used to show an element that is previously hidden:


      $(document).ready(function(){
        $("#btn2").click(function() {
          $("#div2").show("slow");
        });
      });

Example: toggle()

The toggle() method can be used to hide and show an element using one button:


      $(document).ready(function(){
        $("#btn3").click(function() {
          $("#div3").toggle("slow");
        });
      });
