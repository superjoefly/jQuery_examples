Traversing Filtering

The following 3 methods allow you to select a specific element based on its position in a group of elements:

  • first() - returns the first element of the selected elements
  • last() - returns the last element of the selected elements
  • eq() - returns an element with a specific index

Example: first()

Target the first element of the selected elements using the first() method:

  • First list item
  • Second list item
  • Third list item
  • Last list item

      $(document).ready(function(){
        $("#list1 li").first().css("background-color", "yellow");
      });

Example: last()

Target the last element of the selected elements using the last() method:

  • First list item
  • Second list item
  • Third list item
  • Last list item

      $(document).ready(function(){
      $("#list2 li").last().css("background-color", "lime");
      });

Example: eq()

Target the element using a specified index number with the eq() method:

  • First list item
  • Second list item (has index of 1)
  • Third list item
  • Last list item

      $(document).ready(function(){
      $("#list3 li").eq(1).css("background-color", "red");
      });

The following 2 filtering methods allow you to select elements that match, or do not match, a certain criteria:

  • filter() - lets you specify a criteria; elements that do not match the criteria are removed from the selection; thos that do match are returned
  • not() - returns all elements that do NOT match the criteria; this is the opposite of the filter() method

Example: filter()

Target the elements that fit a specific crieteria using the filter() method:

  • First list item
  • Second list item
  • Third list item
  • Last list item

      $(document).ready(function(){
        $("#list4 li").filter(".demo").css("background-color", "purple");
      });

Example: not()

Target the elements that do NOT fit the specified criteria usint the not() method:

  • First list item
  • Second list item
  • Third list item
  • Last list item

      $(document).ready(function(){
        $("#list5 li").not(".demo2").css("background-color", "orange");
      });
