Traversig Siblings

With jQuery you can traverse sideways in the DOM tree to find siblings of an element.

The following methods are available for traversing sibling elements:

  • siblings() - returns all sibling elements; you can also use a parameter to filter the search for siblings
  • next() - returns the next sibling element
  • nextAll() - returns ALL next sibling elements
  • nextUntil() - returns all next sibling elements between two given arguments
  • prev() - returns the previous sibling element
  • prevAll() - returns ALL previous sibling elements
  • prevUntil() - returns all previous sibling elements between two given arguments.

The prev(), prevAll() and prevUntil() methods work the same as the methods below, except with reverse functionality.

Example: siblings()

Target the siblings (red) of an element (first paragraph) using the siblings() method:
