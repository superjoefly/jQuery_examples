Traversing Ancestors

With jQuery you can traverse up the DOM tree to find ancestors of an element.

The following methods can be used to traverse the DOM tree:

  • parent() - returns the direct parent element of the selected element (traverses a single level up the DOM tree)
  • parents() - returns all ancestor elements of the selected element (traverses all the way up to the document's root element)
  • parentsUntil() - returns all ancestor elements between two given arguments

Example: parent()

Target the parent element (blue) using the parent() method: (span.parent() blue)
