Set Content and Attributes

Three simple methods for DOM manipulation are:

  • text() - sets or returns the text content of selected elements
  • html() - sets or returns the content of selected elements (including HTML markup)
  • val() - sets or returns the value of form fields
  • attr() - sets or returns the attribute value of an element

Example: set attribute value with attr()

Mouse over link after clicking the button to see that the href has changed:

Bing.com


      $(document).ready(function(){
        $("#btn6").click(function() {
          $("#test6").attr("href", "http://www.google.com");
        });
      });

Example: set multiple attribute values with attr()

Mouse over the link after clicking the button to see that the href and title have been changed:

Bing.com


      $(document).ready(function(){
        $("#btn7").click(function() {
          $("#test7").attr({
            "href" : "http://www.google.com",
            "title" : "View Google.com"
          });
        });
      });

Example: set attribute value with callback function

Mouse over the link after clicking the button to see that the value of the href has been changed:

Bing.com


      $(document).ready(function(){
        $("#btn8").click(function() {
          $("#test8").attr("href", function(i, origValue) {
            return origValue + "/images";
          });
        });
      });
