Dimensions

jQuery has the following methods for working with dimensions:

  • width() - sets or returns the width of an element (excluding padding, border and margin)
  • height() - sets or returns the height of an element (excluding padding, border and margin)
  • innerWidth() - returns the width of an element (including padding)
  • innerHeight() - returns the height of an element (including padding)
  • outerWidth() - returns the width of an element (including padding and border)
  • outerHeight() - returns the height of an element (including padding and border)
  • outerWidth(true) - returns the width of an element (including padding, border and margin)
  • outerHeight(true) - returns the height of an element (including padding, border and margin)

Example: width() and height()

Return the width and height of an element (excludes padding, border and margin):



      $(document).ready(function(){
        $("#btn1").click(function() {
          var txt = "";
          txt += "Width: " + $("#div1").width() + "
";
          txt += "Height: " + $("#div1").height();
          $("#div1").html(txt);
        });
      });

Example: innerWidth() and innerHeight()

      $(document).ready(function(){

      });

Example: outerWidth() and outerHeight()

Return the outer width and outer height of an element (includes padding and border):



      $(document).ready(function(){

";

      $(document).ready(function(){
        $("#btn4").click(function() {
          var txt = "";

";

      });

Example: set width() and height()



      $(document).ready(function(){
        $("#btn5").click(function() {
          $("#div5").width(400).height(200);
        });
      });
