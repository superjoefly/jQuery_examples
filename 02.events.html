Events

All the different actions that a webpage can respond to are called events. An event represents the precise moment when something happens. Examples of events include:

  • moving a mouse over an element
  • selecting an input field
  • clicking on an element

Example: click()

Click on an element to make it disappear:

Click on me an I will disappear!

      $(document).ready(function(){
        $("#div1").click(function() {
          $(this).hide();
        });
      });

Example: dblclick()

Double click an element to make it disappear:

Double click on me and I will disappear!

      $(document).ready(function(){
        $("#div2").dblclick(function() {
          $(this).hide();
        });
      });

Example: mouseenter()

Enter the element with the cursor to display a popup:

Enter the cursor into this element!

      $(document).ready(function(){
        $("#div3").mouseenter(function() {
          alert("You entered the element!");
        });
      });

Example: mouseleave()

Enter, and then leave the element to display a popup!

Enter and then leave this element to see the popup!

      $(document).ready(function(){
        $("#div4").mouseleave(function() {
          alert("You left the element!");
        });
      });

Example: mousedown()

Disply an alert on mousedown (takes place before releasing the mouse button):

Click the mouse button down over me to display a popup!

      $(document).ready(function(){
        $("#div5").mousedown(function() {
          alert("Mousedown over the element!");
        });
      });

Example: mouseup()

Display an alert on mouseup (takes place after releasing the mouse button):

Click and release the mouse button over me to display an alert!

      $(document).ready(function(){
        $("#div6").mouseup(function() {
          alert("Mouseup over the element!");
        });
      });

Example: hover() (mouseenter/mouseleave)

Display an alert when hovering over, and after leaving, the element:

Hover over me to display an alert!

      $(document).ready(function(){
        $("#div7").hover(function() {
          alert("You hovered over the element!");
        },
      function() {
        alert("You left the element!");
      });
      });
