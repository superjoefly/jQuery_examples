Traversing Descendants

With jQuery you can traverse down the DOM tree to find descendants of an element.

Two available methods for traversing down the DOM tree include:

  • children() - The children method returns all direct children of the selected element. This method only traverses down the DOM tree a single level.
  • find() - This method returns descendant elements of the selected element, all the way down to the last descendant.

Example: children()

Target the direct-child of an element using the children() method: (children() orange)
