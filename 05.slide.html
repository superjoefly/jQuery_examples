Slide

The jQuery slide() method slides elements up and down.

jQuery has the following slide() methods:

  • slideDown()
  • slideUp()
  • slideToggle()

Example: slideDown()

The slideDown() method is used to slide an element down:

Click to slide down the panel!
Well, hello there!

      $(document).ready(function(){
        $("#div1").click(function() {
          $("#panel1").slideDown("slow");
        });
      });

Example: slideUp()

The slideUp() method is used to slide up an element:

Click to slide up the panel!
No!!!! Please Dont!!!

      $(document).ready(function(){
        $("#div2").click(function() {
          $("#panel2").slideUp("slow");
        });
      });

Example: slideToggle()

The slideToggle() method can be used to toggle the slideDown and slideUp methods:

Click to toggle the visibility of the panel!
Hi, again! / Bye, again!

      $(document).ready(function(){
        $("#div3").click(function() {
          $("#panel3").slideToggle("slow");
        });
      });
