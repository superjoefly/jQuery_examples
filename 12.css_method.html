Get and Set Using css() Method

The css() method sets or returns one or more style properties for the selected elements.

Example: css() return

Use the css() method to return the value of a specified CSS property from the FIRST matched element:

This is a paragraph!

This is another paragraph!

And yet, another paragraph!


      $(document).ready(function(){
        $("#btn1").click(function() {
          alert("background-color: " + $("#test1").css("background-color"));
        });
      });

Example: css() set

Use the css() method to set a specific CSS property for ALL matched elements:

This is a paragraph!

This is another paragraph!

And yet, another paragraph!


      $(document).ready(function(){
        $("#btn2").click(function() {
          $("#test2*").css("background-color", "yellow");
        });
      });

Example: css() multiple

Use the css() method to set multiple CSS properties for all matched elements:

This is a paragraph!

This is another paragraph!

And yet, another paragraph!


      $(document).ready(function(){
        $("#btn3").click(function() {
          $("#test3*").css({
            "background-color":"lime", "font-size":"22px", "border": "2px solid black"});
        });
      });
